3 задание

3.1 Как проверить, что данный файл есть в текущем каталоге?
Можно использовать команду ls, которая выведем все файлы текущего каталога, по выведенному списку файлов, можно отследить, если файл с таким названием или нет.
Но если файлов много, но можно использовать функцию " find [название файла]", которая должна будет просто вывести имя файла в случае, если он найден, или ошибку (в противном случае).

3.2 Как установить владельцем файла пользователя user группы users?
Нужно запустить команду " chmod user [путь к файлу, владельца которого нужно изменить]" с правами администратора, то есть добавить перед командой sudo.

3.3 Как установить следующие права достпа...
$ chmod 750 [путь к файлу]
